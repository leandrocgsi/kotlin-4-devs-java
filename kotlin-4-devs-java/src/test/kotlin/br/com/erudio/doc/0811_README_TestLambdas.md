# Find the shortest student name

## Instructions

Given an array of students, implement a function which returns the student that is the shortest name.

## Examples

```kotlin
    val students = listOf(
        Student("Ada", 19),
        Student("Manoel", 23),
        Student("Maria", 20),
        Student("João", 39),
        Student("José", 16)
    )

    findTheShortestStudentName(students)?.name shouldBeEqualTo "Ada"
```
